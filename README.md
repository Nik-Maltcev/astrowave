# Astro Wave

Персональные расчеты матрицы судьбы и натальной карты с AI-анализом.

## Возможности

- **Личная матрица** - базовый расчет матрицы судьбы и натальной карты
- **Детская матрица** - анализ талантов и потенциала ребенка
- **Матрица совместимости** - оценка совместимости двух людей
- **Финансовая матрица** - рекомендации по заработку и инвестициям

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `npm install`
3. Создайте файл `.env` на основе `.env.example`
4. Добавьте ваш DeepSeek API ключ в `.env`
5. Запустите: `npm start`

## Деплой на Railway

1. Подключите репозиторий к Railway
2. Добавьте переменную окружения `DEEPSEEK_API_KEY`
3. Railway автоматически задеплоит проект

## API ключ DeepSeek

Получите бесплатный API ключ на https://platform.deepseek.com/

## Технологии

- Express.js
- DeepSeek API (модель deepseek-chat)
- Vanilla JavaScript
- CSS Grid & Flexbox
